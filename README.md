# Tasks
Scripts to make Matias life easier.

Reason for uploading: in case Matias loses it.

##Example of what it does:

#### It processes IPs in this format:
- 10.8.1.50	0
- 10.8.1.51	0
- 10.142.125/4123,4124,2441
- 10.234.423/6345,6234
- 10.524.562/5234,7253,1634
- 10.8.1.50	4123,4124,2441
- 10.344.421/5123
- 10.412.412

#### And converts them to:
- 10.8.1.50/0
- 10.8.1.51/0
- 10.142.125/4123
- 10.142.125/4124
- 10.142.125/2441
- 10.234.423/6345
- 10.234.423/6234
- 10.524.562/5234
- 10.524.562/7253
- 10.524.562/1634
- 10.8.1.50/4123
- 10.8.1.50/4124
- 10.8.1.50/2441
- 10.344.421/5123
- 10.412.412
